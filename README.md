Лендинг iOS приложения
Минималистичный лендинг с тарифами и локализацией для iOS приложения.

Что внутри
Два тарифа — базовый и премиум с визуальным выбором

Локализация — автоматически определяет язык системы (рус/англ)

Анимации — плавные CSS переходы и hover-эффекты

Адаптив — оптимизирован под iPhone и iPad

Развертывание — готовая конфигурация для Vercel

Структура
text
index.html # Главная страница
i18n/ # Локализация (en, pt, fr, ja, de,es)
assets/ # Изображения
css/ # Стили
js/ # Основная логика и локализация

Быстрый старт
Замени тексты в i18n/

Обнови тарифы в data-i18n-params

Задеплой на Vercel:

bash
npm i -g vercel
vercel --prod

Технологии
Чистый HTML/CSS/JS
CSS Grid & Flexbox
Fetch API для локализации
